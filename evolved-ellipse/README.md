# Print Your Design Static Site

Static Astro landing page implementation based on your selected UI direction.

## Implemented Requirements

- Upload Your Design CTA opens the user's email client using a `mailto:` URL.
- Dedicated test recipient is prefilled in the `To` field.
- Precision Workflow section removes the "Choose Material" step.

## Run Locally

```sh
npm install
npm run dev
```

## Build For Production

```sh
npm run build
npm run preview
```

## Main Page File

- `src/pages/index.astro`
